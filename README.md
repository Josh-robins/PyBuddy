# PyBuddy – Your AI-Powered Python Tutor

An interactive, web-based Python tutoring chatbot powered by Google's Gemini AI. PyBuddy explains Python concepts clearly, provides working code examples, and helps learners at all levels—from complete beginners to intermediate programmers.

## 🚀 Features

- **Interactive Chat** - Ask anything about Python, from basics to advanced concepts
- **Real-time Code Examples** - Get runnable, well-commented Python code
- **Syntax Highlighting** - Code blocks are highlighted for better readability
- **Copy Code Button** - Easily copy code examples to your clipboard
- **Chat History** - Save and manage multiple conversations using the sidebar
- **Dark/Light Theme** - Toggle between dark and light modes
- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **Fast & Reliable** - Powered by Google Gemini 3 Flash Preview model

## 🎯 What PyBuddy Can Help With

- Python basics (variables, data types, operators)
- Functions, OOP, and design patterns
- Popular libraries (NumPy, Pandas, Flask, etc.)
- Debugging and code reviews
- Project ideas and learning roadmaps
- Coding challenges and quizzes

## 📋 Quick Start

1. **Open in Browser** - Simply open `pybuddy.html` in any modern web browser
2. **Start Chatting** - Type your Python question and hit Enter or click Send
3. **Save Conversations** - All chats are automatically saved in your browser's local storage
4. **Load Previous Chats** - Click any saved chat in the sidebar to resume

## 🔧 How It Works

- Uses **Google Gemini API** for intelligent responses
- Stores conversations locally using browser localStorage
- Features **Inter font** for clean, modern typography
- **Markdown rendering** for formatted responses with code syntax highlighting

## 🛠️ Technical Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **AI Model**: Google Gemini 3 Flash Preview
- **Highlighting**: Highlight.js
- **Markdown**: Marked.js
- **Fonts**: Inter (system font fallbacks)

## 📦 Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for Gemini API calls)
- Valid Google Gemini API key

## ⚙️ Setup Instructions

1. **Get a Gemini API Key**
   - Visit [Google AI Studio](https://aistudio.google.com/)
   - Generate a new API key
   - The file includes the API key (keep it secure in production!)

2. **Run Locally**
   - Download `pybuddy.html`
   - Open in your web browser
   - Start chatting immediately

3. **Deploy on Web**
   - Upload to any static hosting (GitHub Pages, Netlify, Vercel)
   - No backend server required
   - All processing happens in your browser
